## Меню бортового компьютера

!!! info ""

       Только для комплектаций с системой Sensing.

!!! info ""

       Настройка выполняется при выставленном селекторе коробки передач в положении ++"P"++ с помощью кнопок
       ++"▲"++ ++"▼"++ и ++"SEL/RESET"++ справа на руле.

![Настройка](/assets/setup/boar-computer-setup.jpg)

Выберите пункт меню ++"時計/車両設定"++ (Настройка часов / Настройка
автомобиля) на мультиинформационном дисплее.
